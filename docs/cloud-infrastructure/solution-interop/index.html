    Solution Interoperability of Cloud Infrastructure

    Solution Interoperability of Cloud Infrastructure #

    Integrate the Cloud Infrastructure validated solution with components added to your VMware Cloud Foundation environment by other validated solutions for operations management and business continuity. You can use such validated solutions for logging, backup and restore, disaster recovery, and life cycle management with certain considerations.

    This validated solution includes guidance only for implementing intelligent operations management. Performing all deployments and configurations for operations management and business continuity are out of scope. Such guidance is part of other validated solutions. However, this solution provides either design or implementation guidance to enable the integration with such solutions.

    Monitoring and Alerting for Cloud Infrastructure #

    After you implement the Cloud Infrastructure validated solution, configure both monitoring and alerting for the vRealize Automation components in the VMware Cloud Foundation environment.[Read more]

    Logging for Cloud Infrastructure #

    After you implement the Cloud Infrastructure validated solution, by using VMware or third-party components, collect log data in a central place from the components that are newly-added to or re-configured in your VMware Cloud Foundation environment.[Read more]

    Advanced Load Balancing for Cloud Infrastructure #

    After you implement the Cloud Infrastructure validated solution, by using VMware or third-party components, enable load balancing of the components that are newly-added to your VMware Cloud Foundation environment.[Read more]

    Life Cycle Management of Cloud Infrastructure #

    After you implement the Cloud Infrastructure validated solution, by using VMware or third-party components, enable upgrade and patching of the components that are newly-added to your VMware Cloud Foundation environment. [Read more]
