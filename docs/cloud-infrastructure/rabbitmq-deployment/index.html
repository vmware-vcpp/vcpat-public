    Implementation of RabbitMQ includes deployment of VMware Cloud Director on VMware Cloud Foundation using UI of VMware Cloud Provider Lifecycle Manager

    For information on the VMware Cloud Provider Lifecycle Manager design, refer to the Detailed design of RabbitMQ section.

    Prerequisites #

    Verify that your environment is configured according to  Before You Apply This Guidance  and the Planning and Preparation of RabbitMQ section.
